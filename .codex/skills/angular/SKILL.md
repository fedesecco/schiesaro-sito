# Angular Skill

## When to use

Use this skill for Angular work in this repository, especially when the task involves:

- Angular architecture, APIs, or framework behavior
- Angular CLI usage and workspace-aware operations
- Angular Material or CDK integration
- Modernization toward current Angular patterns
- Finding canonical Angular examples or best practices

## Workflow

1. Prefer the `angular-cli` MCP server for Angular-specific questions and operations.
2. Use MCP documentation and best-practices tools before answering from memory when Angular behavior is involved.
3. Keep implementations aligned with the existing workspace and avoid introducing outdated Angular patterns.
4. Prefer standalone components, modern template control flow, typed APIs, and current Angular Material patterns where they fit the codebase.

## Project context

- This repository uses Angular 20.x.
- `@angular/material` and `@angular/cdk` are installed.
- Keep changes consistent with the current workspace structure and CLI configuration.
